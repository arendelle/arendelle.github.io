Arendelle Kit

Arendelle Kit is a set of special tools designed to empower Arendelle Masters and Arendelle developers to work with the language in a higher level. This tools are many from the tools to generated Arendelle coded texts to the ones like folded and highlighted HTML code generator. This tools are gathered together in a collection called Arendelle Kit available to use via our developer system



Developer Front-Ends

Front-ends of Arendelle are divided to two major types: General Purpose User Interfaces, Developer Interfaces. The group one is of course our official apps and desktop applications but the type two is the special interfaces designed for Arendelle Masters, we have command-line REPL systems, Swift Playground like real-time interactive environment and much more that you can use to test and debug special Arendelle codes or you can use them to deploy Arendelle on new systems and services.
